Leukippos-SynBioAppSelector
===========================

An app that serves as an index of available software used in synthetic biology, categorizing them by function. Being developed by the team at the Leukippos Institute, leukippos.org. For more information, see: SynBio App Selector: an Application Selection Engine forSynthetic Biology.
